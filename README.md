# XML-Lint

## Description  
Run xmllint on the sdlxliff, xml and xlf files.  
The types of files that will be accepted as uploads are sdlrpx, xlz, wsxz and zip.  
\* Upload file size is limited to 200MB  

## Requires  
- xmllint
- zip command
- unzip command
